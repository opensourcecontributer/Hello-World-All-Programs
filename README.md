### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/opensourcecontributer/Hello-World-All-Programs.git
   cd Hello-World-All-Programs
   ```

2. **Open the project**
   ```bash
   # Simply open index.html in your browser
   open index.html
   # OR use a local server (recommended)
   python -m http.server 8000
   # OR if you have Node.js
   npx serve .
   ```

3. **Start exploring!**
   - Browse through different programming languages
   - Toggle between dark and light themes
   - Use the search functionality to find specific languages
   - Click on any language button to view its Hello World program on GitHub

### ğŸ“‹ Contribution Steps

1. **Fork the repository**
   ```bash
   # Click the "Fork" button on GitHub
   ```

2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/Hello-World-All-Programs.git
   cd Hello-World-All-Programs
   ```

3. **Create a new branch**
   ```bash
   git checkout -b feature/add-[language-name]
   # OR
   git checkout -b fix/issue-description
   ```

4. **Make your changes**
   - Add your Hello World program
   - Update index.html if adding a new language
   - Test your changes locally

5. **Commit and push**
   ```bash
   git add .
   git commit -m "Add Hello World program in [Language Name]"
   git push origin feature/add-[language-name]
   ```

6. **Create a Pull Request**
   - Go to your fork on GitHub
   - Click "New Pull Request"
   - Fill out the PR template
   - Wait for review and merge!
